# Restore the serialized emulator

This function restores the serialized emulator created by
[`serialize()`](http://mingdeyu.github.io/dgpsi-R/dev/reference/serialize.md).

## Usage

``` r
deserialize(object)
```

## Arguments

- object:

  the serialized object of an emulator.

## Value

The S3 class of a GP emulator, a DGP emulator, a linked (D)GP emulator,
or a bundle of (D)GP emulators.

## Details

See further examples and tutorials at
<https://mingdeyu.github.io/dgpsi-R/dev/>.

## Note

See the *Note* section in
[`serialize()`](http://mingdeyu.github.io/dgpsi-R/dev/reference/serialize.md).
