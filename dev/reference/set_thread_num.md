# Set the number of threads

This function sets the number of threads for parallel computations
involved in the package.

## Usage

``` r
set_thread_num(num)
```

## Arguments

- num:

  the number of threads. If it is greater than the maximum number of
  threads available, the number of threads will be set to the maximum
  value.

## Value

No return value.
