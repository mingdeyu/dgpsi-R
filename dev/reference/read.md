# Load the stored emulator

This function loads the `.pkl` file that stores the emulator.

## Usage

``` r
read(pkl_file)
```

## Arguments

- pkl_file:

  the path to and the name of the `.pkl` file where the emulator is
  stored.

## Value

The S3 class of a GP emulator, a DGP emulator, a linked (D)GP emulator,
or a bundle of (D)GP emulators.
